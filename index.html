# EJERCICIO
1 Debes crear una función que permita retornar una lista de usuarios (longitud de la lista, es a criterio propio)
2 Una función que permita retornar un usuario en específico detallando sus propiedades, Se agrego al ejercicio 2 tablas una llamada "movimiento" en donde se almacenan los movimientos (recargas,tranferencias realizadas,transferencia recibida) y otros datos relacionados con las mismas como fecha, hora, de haber realizado una tranferencia se almacena id del usuario que la recibio y el monto, ademas en cada registro queda almacenado el saldo anterior y el posterior al movimiento.
3 Crea una función que permita recargar el saldo de la cuenta de un usuario.
4 Necesitamos hacer una transferencia de un usuario a otro, crea una función que me permita enviar dinero este dinero.
